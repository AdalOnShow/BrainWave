## ğŸ“ About the Project

Brainwave is a frontend-focused application that highlights interactive UI elements, smooth navigation, and modern styling.  
Built with React Router DOM for dynamic routing and enhanced with parallax effects using `react-just-parallax`, the project offers a visually appealing browsing experience.

The goal was to create a lightweight, fast, and user-friendly interface that adapts well across all devices.
